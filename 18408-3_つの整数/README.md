# 문제
3 つの整数 A, B, C が与えられる．A, B, C はそれぞれ 1 または 2 である．1 と 2 のうち，どちらが多くあるか．

# 입력
入力は以下の形式で標準入力から与えられる．

A B C
# 출력
1 と 2 のうち，個数が多い方を出力せよ．

# 제한
A, B, C はそれぞれ 1 または 2 である．
# 예제 입력 1 
1 2 1
# 예제 출력 1 
1  
1 が 2 個で 2 が 1 個なので，個数の多い 1 を出力する．
# 예제 입력 2 
2 2 2
# 예제 출력 2 
2  
すべて 2 なので，2 を出力する．
